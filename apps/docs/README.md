# TGB Form Documentation

Starlight documentation for `@tgb-form/core`, `@tgb-form/react`, and `@tgb-form/vue`.

```bash
pnpm --filter @tgb-form/docs dev
pnpm --filter @tgb-form/docs typecheck
pnpm --filter @tgb-form/docs build
```

The site is statically generated for GitHub Pages. The build generates API pages with TypeDoc and Open Graph images with Takumi.
