`@tgb-form/react` connects portable definitions to React components. The definition stays JSON-safe; renderers stay in application code.

[Open the React package on GitHub](https://github.com/VueEasyForm/tgb-form/tree/main/packages/react).

## Supported Flows

`TgbForm` supports three usage levels:

1. Implicit: `TgbForm` creates the TanStack form instance and reads renderers from `TgbFormContext`.
2. Hybrid: you provide either the TanStack form instance or the renderer registry, while `TgbForm` supplies the rest.
3. Fully explicit: you provide both `instance` and `renderers` directly to `TgbForm`.

## Quick Start

```tsx
import { defineForm, FieldDataType, ValidationRuleKind } from '@tgb-form/core';
import {
  BaseReactRendererProps,
  TgbForm,
  TgbFormContext,
  createReactRendererRegistry,
} from '@tgb-form/react';

const definition = defineForm({
  fields: {
    email: {
      type: FieldDataType.String,
      defaultValue: '',
      label: 'Email',
      component: 'email-input',
      props: { placeholder: 'you@example.com' },
      rules: [{ kind: ValidationRuleKind.Email, message: 'Enter a valid email' }],
    },
    subscribed: {
      type: FieldDataType.Boolean,
      defaultValue: true,
      label: 'Subscribe',
    },
  },
});

function TextInput({ field, label, name, props, value, errors }: BaseReactRendererProps) {
  return (
    <label>
      <span>{label}</span>
      <input
        name={name}
        value={String(value ?? '')}
        placeholder={String(props?.placeholder ?? '')}
        onBlur={() => field.handleBlur()}
        onChange={(event) => field.handleChange(event.currentTarget.value)}
      />
      {errors.length > 0 ? <small>{errors.map(String).join(', ')}</small> : null}
    </label>
  );
}

function CheckboxInput({ field, label, name, value }: BaseReactRendererProps) {
  return (
    <label>
      <input
        name={name}
        type="checkbox"
        checked={Boolean(value)}
        onChange={(event) => field.handleChange(event.currentTarget.checked)}
      />
      <span>{label}</span>
    </label>
  );
}

const renderers = createReactRendererRegistry({
  byName: {
    'email-input': TextInput,
  },
  byType: {
    [FieldDataType.String]: TextInput,
    [FieldDataType.Boolean]: CheckboxInput,
  },
});

export function NewsletterForm() {
  return (
    <TgbFormContext renderers={renderers}>
      <TgbForm
        definition={definition}
        tanstackOptions={{
          onSubmit: async ({ value }) => {
            console.log(value);
          },
        }}
      >
        <button type="submit">Submit</button>
      </TgbForm>
    </TgbFormContext>
  );
}
```

That is the implicit flow: application code provides a renderer registry once through context, and `TgbForm` creates the TanStack form instance from `definition` and `tanstackOptions`.

## The TgbForm Component

`TgbForm`:

- Calls `toTanStackOptions(definition, tanstackOptions)` when no `instance` is passed.
- Creates a TanStack React Form instance with `useForm`.
- Provides the form instance to child fields.
- Renders fields by `order`, then declaration order.
- Calls `form.handleSubmit()` on submit.

`TgbFormField` is the automatic registry-based resolver. It wraps TanStack's `Field`, resolves the renderer for one field definition, and passes TanStack field bindings and state into that renderer.

Pass `fields` to render a named subset:

```tsx
import { defineForm, FieldDataType } from '@tgb-form/core';
import { TgbForm, createReactRendererRegistry } from '@tgb-form/react';

const definition = defineForm({
  fields: {
    email: {
      type: FieldDataType.String,
      defaultValue: '',
    },
  },
});

const TextInput = () => null;

const renderers = createReactRendererRegistry({
  byType: {
    [FieldDataType.String]: TextInput,
  },
});

<TgbForm
  definition={definition}
  fields={['email']}
  renderers={renderers}
/>;
```

## Hybrid Flow

Pass local renderers while still letting `TgbForm` create the TanStack form instance:

```tsx
import { defineForm, FieldDataType } from '@tgb-form/core';
import { TgbForm, createReactRendererRegistry } from '@tgb-form/react';

const definition = defineForm({
  fields: {
    email: {
      type: FieldDataType.String,
      defaultValue: '',
    },
  },
});

const TextInput = () => null;

const renderers = createReactRendererRegistry({
  byType: {
    [FieldDataType.String]: TextInput,
  },
});

<TgbForm
  definition={definition}
  renderers={renderers}
  tanstackOptions={{
    onSubmit: async ({ value }) => {
      console.log(value);
    },
  }}
/>;
```

Or pass an existing TanStack form instance while still reading renderers from context:

```tsx
import { defineForm, FieldDataType, toTanStackOptions } from '@tgb-form/core';
import { TgbForm, TgbFormContext, createReactRendererRegistry } from '@tgb-form/react';
import { useForm } from '@tanstack/react-form';

const definition = defineForm({
  fields: {
    email: {
      type: FieldDataType.String,
      defaultValue: '',
    },
  },
});

const TextInput = () => null;

const renderers = createReactRendererRegistry({
  byType: {
    [FieldDataType.String]: TextInput,
  },
});

function NewsletterForm() {
  const form = useForm({
    ...toTanStackOptions(definition),
    onSubmit: async ({ value }) => {
      console.log(value);
    },
  });

  return (
    <TgbFormContext renderers={renderers}>
      <TgbForm
        definition={definition}
        instance={form}
      />
    </TgbFormContext>
  );
}
```

## Fully Explicit Flow

Pass an existing TanStack form instance and local renderers when you want full control:

```tsx
import { defineForm, FieldDataType, toTanStackOptions } from '@tgb-form/core';
import { TgbForm, createReactRendererRegistry } from '@tgb-form/react';
import { useForm } from '@tanstack/react-form';

const definition = defineForm({
  fields: {
    email: {
      type: FieldDataType.String,
      defaultValue: '',
    },
  },
});

const TextInput = () => null;

const renderers = createReactRendererRegistry({
  byType: {
    [FieldDataType.String]: TextInput,
  },
});

function NewsletterForm() {
  const form = useForm({
    ...toTanStackOptions(definition),
    onSubmit: async ({ value }) => {
      console.log(value);
    },
  });

  return (
    <TgbForm
      definition={definition}
      instance={form}
      renderers={renderers}
    />
  );
}
```

## Renderer Props

Every React renderer receives:

| Prop          | Description                                         |
| ------------- | --------------------------------------------------- |
| `name`        | Field name from the definition map.                 |
| `field`       | TanStack field binding.                             |
| `form`        | Current TanStack form instance.                     |
| `label`       | Field label, if present.                            |
| `description` | Field description, if present.                      |
| `props`       | JSON-safe renderer props from the field definition. |
| `value`       | Current field value.                                |
| `errors`      | Current field errors.                               |

Renderer components should translate these props into the input components used by your app.

## Context And Overrides

Use `TgbFormContext` for app-wide renderers:

```tsx twoslash
import { defineForm, FieldDataType } from '@tgb-form/core';
import { TgbForm, TgbFormContext, createReactRendererRegistry } from '@tgb-form/react';

const definition = defineForm({
  fields: {
    email: {
      type: FieldDataType.String,
      defaultValue: '',
    },
  },
});

const TextInput = () => null;

const renderers = createReactRendererRegistry({
  byType: {
    [FieldDataType.String]: TextInput,
  },
});

<TgbFormContext renderers={renderers}>
  <TgbForm definition={definition} />
</TgbFormContext>;
```

Use the `renderers` prop for a local override:

```tsx twoslash
import { defineForm, FieldDataType } from '@tgb-form/core';
import { TgbForm, createReactRendererRegistry } from '@tgb-form/react';

const definition = defineForm({
  fields: {
    email: {
      type: FieldDataType.String,
      defaultValue: '',
    },
  },
});

const TextInput = () => null;

const localRenderers = createReactRendererRegistry({
  byType: {
    [FieldDataType.String]: TextInput,
  },
});

<TgbForm
  definition={definition}
  renderers={localRenderers}
/>;
```

Renderer resolution follows core rules: explicit `component` names resolve through `byName`; fields without `component` fall back to `byType[field.type]`.
