TGB Form does not replace TanStack Form. It generates the parts TanStack Form can derive from a portable definition: default values and submit validation.

[Open the TanStack bridge on GitHub](https://github.com/VueEasyForm/tgb-form/blob/main/packages/core/src/tanstack.ts).

## What

`toTanStackOptions` returns:

- `defaultValues` from each field's `defaultValue`.
- `validators.onSubmit` from the compiled Valibot schema.
- any extra options you pass through.

```ts twoslash
import { defineForm, FieldDataType, toTanStackOptions, ValidationRuleKind } from '@tgb-form/core';

const newsletterForm = defineForm({
  fields: {
    email: {
      type: FieldDataType.String,
      defaultValue: '',
      label: 'Email',
      rules: [
        { kind: ValidationRuleKind.Required, message: 'Email is required' },
        { kind: ValidationRuleKind.Email, message: 'Enter a valid email' },
      ],
    },
    subscribed: {
      type: FieldDataType.Boolean,
      defaultValue: false,
      label: 'Subscribe',
    },
  },
});

const options = toTanStackOptions(newsletterForm, {
  formId: 'newsletter',
});
```

## Why

The form definition already knows initial values and submit rules. Reusing that data keeps builders, stored schemas, validation, and rendered forms aligned.

TanStack Form still owns state, subscriptions, submission, and framework hooks.

## How

Use generated options with a TanStack framework hook:

```tsx
import { defineForm, FieldDataType, toTanStackOptions, ValidationRuleKind } from '@tgb-form/core';
import { useForm } from '@tanstack/react-form';

const newsletterForm = defineForm({
  fields: {
    email: {
      type: FieldDataType.String,
      defaultValue: '',
      rules: [
        { kind: ValidationRuleKind.Required, message: 'Email is required' },
        { kind: ValidationRuleKind.Email, message: 'Enter a valid email' },
      ],
    },
    subscribed: {
      type: FieldDataType.Boolean,
      defaultValue: false,
    },
  },
});

async function subscribe(email: string, subscribed: boolean) {
  console.log({ email, subscribed });
}

export function NewsletterForm() {
  const form = useForm({
    ...toTanStackOptions(newsletterForm),
    onSubmit: async ({ value }) => {
      await subscribe(value.email, value.subscribed);
    },
  });

  return (
    <form
      onSubmit={(event) => {
        event.preventDefault();
        void form.handleSubmit();
      }}
    >
      <form.Field name="email">
        {(field) => (
          <input
            name={field.name}
            value={field.state.value}
            onBlur={field.handleBlur}
            onChange={(event) => field.handleChange(event.target.value)}
          />
        )}
      </form.Field>
      <button type="submit">Subscribe</button>
    </form>
  );
}
```

Or let an adapter create the form instance:

```tsx
import { defineForm, FieldDataType, ValidationRuleKind } from '@tgb-form/core';
import { TgbForm, createReactRendererRegistry } from '@tgb-form/react';

const newsletterForm = defineForm({
  fields: {
    email: {
      type: FieldDataType.String,
      defaultValue: '',
      rules: [
        { kind: ValidationRuleKind.Required, message: 'Email is required' },
        { kind: ValidationRuleKind.Email, message: 'Enter a valid email' },
      ],
    },
    subscribed: {
      type: FieldDataType.Boolean,
      defaultValue: false,
    },
  },
});

const TextInput = () => null;
const CheckboxInput = () => null;

const renderers = createReactRendererRegistry({
  byType: {
    [FieldDataType.String]: TextInput,
    [FieldDataType.Boolean]: CheckboxInput,
  },
});

async function subscribe(email: string, subscribed: boolean) {
  console.log({ email, subscribed });
}

<TgbForm
  definition={newsletterForm}
  renderers={renderers}
  tanstackOptions={{
    onSubmit: async ({ value }) => {
      await subscribe(value.email, value.subscribed);
    },
  }}
/>;
```

## Defaults Only

Use `getDefaultValues` when you need defaults without a validation schema:

```ts twoslash
import { defineForm, FieldDataType, getDefaultValues } from '@tgb-form/core';

const newsletterForm = defineForm({
  fields: {
    email: {
      type: FieldDataType.String,
      defaultValue: '',
    },
    subscribed: {
      type: FieldDataType.Boolean,
      defaultValue: false,
    },
  },
});

const defaultValues = getDefaultValues(newsletterForm);
```
