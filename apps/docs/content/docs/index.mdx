Define forms as data. Render them anywhere.

EasyForm is a fast, flexible wrapper around TanStack Form and Valibot. It keeps form schemas JSON-safe while leaving runtime code free to choose React, Vue, custom renderers, storage, and validation extensions.

## What

EasyForm gives you a portable form definition:

```ts twoslash
import { defineForm, FieldDataType, ValidationRuleKind } from '@easyform/core';

export const newsletterForm = defineForm({
  fields: {
    email: {
      type: FieldDataType.String,
      defaultValue: '',
      label: 'Email',
      component: 'email-input',
      props: { placeholder: 'you@example.com', autocomplete: 'email' },
      rules: [
        { kind: ValidationRuleKind.Required, message: 'Email is required' },
        { kind: ValidationRuleKind.Email, message: 'Enter a valid email' },
      ],
    },
    subscribed: {
      type: FieldDataType.Boolean,
      defaultValue: true,
      label: 'Subscribe',
    },
  },
});
```

The same definition can be serialized, restored, compiled to Valibot, converted to TanStack Form options, and rendered through a registry.

## Why

Most form libraries start from components. That works until the form needs to be stored, edited in a builder, generated from configuration, or rendered in more than one framework.

EasyForm starts from data:

| Need                   | EasyForm layer      |
| ---------------------- | ------------------- |
| Save a form definition | `serializeForm`     |
| Load builder output    | `deserializeForm`   |
| Validate values        | `toValibotSchema`   |
| Use TanStack Form      | `toTanStackOptions` |
| Choose UI components   | renderer registries |

TanStack Form still owns form state. Valibot still owns validation. EasyForm connects them with a schema that can move through storage.

## How

Install the core package and one adapter:

```sh
pnpm add @easyform/core valibot @tanstack/form-core
pnpm add @easyform/react @tanstack/react-form
# or
pnpm add @easyform/vue @tanstack/vue-form
```

Render in React:

```tsx twoslash
import { FieldDataType } from '@easyform/core';
import { EzForm, createReactRendererRegistry, type ReactRendererProps } from '@easyform/react';
import { newsletterForm } from './newsletter-form';

type FieldBinding = {
  handleBlur: () => void;
  handleChange: (value: unknown) => void;
};

function TextInput({
  field,
  label,
  name,
  props,
  value,
  errors,
}: ReactRendererProps<any, FieldBinding>) {
  return (
    <label>
      <span>{label}</span>
      <input
        name={name}
        value={String(value ?? '')}
        placeholder={String(props?.placeholder ?? '')}
        onBlur={() => field.handleBlur()}
        onChange={(event) => field.handleChange(event.currentTarget.value)}
      />
      {errors.length > 0 ? <small>{errors.map(String).join(', ')}</small> : null}
    </label>
  );
}

const renderers = createReactRendererRegistry({
  byName: { 'email-input': TextInput },
  byType: { [FieldDataType.String]: TextInput },
});

export function NewsletterForm() {
  return (
    <EzForm
      definition={newsletterForm}
      renderers={renderers}
      tanstackOptions={{
        onSubmit: async ({ value }) => {
          console.log(value);
        },
      }}
    />
  );
}
```

Store and restore the definition:

```ts twoslash
import { deserializeForm, serializeForm, toTanStackOptions } from '@easyform/core';

const stored = JSON.stringify(serializeForm(newsletterForm));
const restored = deserializeForm(stored);
const options = toTanStackOptions(restored);
```
