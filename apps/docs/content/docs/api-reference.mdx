This page lists the public APIs documented by package.

## Core

[Open public exports on GitHub](https://github.com/VueEasyForm/easyform/blob/main/packages/core/src/index.ts).

### Definitions

| API                                | Description                                                           |
| ---------------------------------- | --------------------------------------------------------------------- |
| `defineForm(definition, options?)` | Parses, validates, normalizes, and clones a form definition.          |
| `serializeForm(form)`              | Returns JSON-safe form data and removes runtime-only registries.      |
| `deserializeForm(input, options?)` | Parses a JSON string or unknown value into a normalized runtime form. |
| `VFormDefinition`                  | Valibot schema for a full form definition.                            |
| `VFieldDefinition`                 | Valibot schema for a single field definition.                         |
| `cloneJson(value)`                 | Deep-clones JSON-safe data.                                           |
| `isJsonObject(value)`              | Checks for a plain JSON object, excluding arrays and `null`.          |

```ts twoslash
import { defineForm, deserializeForm, serializeForm } from '@easyform/core';
```

### Types

| Type                    | Description                                                    |
| ----------------------- | -------------------------------------------------------------- |
| `FormDefinition`        | Normalized JSON-safe form definition.                          |
| `RuntimeFormDefinition` | Normalized form plus optional runtime validator registry.      |
| `FormDefinitionInput`   | Authoring input accepted by `defineForm`.                      |
| `FieldDefinition`       | One JSON-safe field definition.                                |
| `FieldDefinitionInput`  | Authoring input for one field.                                 |
| `JsonValue`             | JSON value accepted in defaults, props, metadata, and options. |
| `JsonObject`            | JSON object used for metadata, props, and validator options.   |

### Enums

| Enum                 | Members                                                                       |
| -------------------- | ----------------------------------------------------------------------------- |
| `FieldDataType`      | `String`, `Number`, `Boolean`, `Object`, `Array`                              |
| `ValidationRuleKind` | `Required`, `MinLength`, `MaxLength`, `Pattern`, `Min`, `Max`, `Email`, `Url` |

### Validators

| API                         | Description                                              |
| --------------------------- | -------------------------------------------------------- |
| `toValibotSchema(form)`     | Compiles a form into a Valibot object schema.            |
| `createValidatorRegistry()` | Creates a registry for named custom validator compilers. |
| `VValidationRule`           | Valibot schema for built-in validation rules.            |
| `VCustomValidatorReference` | Valibot schema for custom validator references.          |

```ts twoslash
import { createValidatorRegistry, toValibotSchema } from '@easyform/core';
```

`createValidatorRegistry()` supports a fluent registration style:

```ts twoslash
import * as v from 'valibot';
import { createValidatorRegistry } from '@easyform/core';

const validators = createValidatorRegistry()
  .register('startsWithA', ({ message }) =>
    v.check((value: string) => value.startsWith('A'), message),
  )
  .register('endsWithZ', ({ message }) => v.check((value: string) => value.endsWith('z'), message));
```

### Renderer Registries

| API                                | Description                                                                     |
| ---------------------------------- | ------------------------------------------------------------------------------- |
| `createRendererRegistry(registry)` | Creates a named and type-based renderer registry while preserving literal keys. |
| `resolveRenderer(field, registry)` | Resolves `field.component` first, then `registry.byType[field.type]`.           |

```ts twoslash
import { createRendererRegistry, resolveRenderer } from '@easyform/core';
```

### TanStack Form

[Open TanStack helpers on GitHub](https://github.com/VueEasyForm/easyform/blob/main/packages/core/src/tanstack.ts).

| API                                 | Description                                           |
| ----------------------------------- | ----------------------------------------------------- |
| `getDefaultValues(form)`            | Extracts cloned default values from top-level fields. |
| `toTanStackOptions(form, options?)` | Returns `defaultValues` and `validators.onSubmit`.    |
| `InferFormValues<TForm>`            | Infers values from each field's `defaultValue`.       |
| `EasyFormTanStackOptions`           | Pass-through options accepted by `toTanStackOptions`. |
| `EasyFormTanStackOutput<TForm>`     | Output type from `toTanStackOptions`.                 |

Twoslash is enabled for TypeScript examples that benefit from inline type inspection:

```ts twoslash
type FieldDefinition<TDefault> = {
  defaultValue: TDefault;
};

type FormDefinition = {
  fields: Record<string, FieldDefinition<unknown>>;
};

type InferFormValues<TForm extends FormDefinition> = {
  readonly [TKey in keyof TForm['fields']]: TForm['fields'][TKey]['defaultValue'];
};

const newsletterForm = {
  fields: {
    email: { defaultValue: '' },
    subscribed: { defaultValue: true },
  },
};

type NewsletterValues = InferFormValues<typeof newsletterForm>;
//   ^?

const values: NewsletterValues = {
  email: 'ada@example.com',
  subscribed: true,
};

values.email;
//     ^?
```

## React

| API                                     | Description                                                                   |
| --------------------------------------- | ----------------------------------------------------------------------------- |
| `EzFormContext`                         | Provides a React renderer registry to descendant forms and fields.            |

| `EzField`                               | Binds one field definition to the current TanStack form instance.             |
| `createReactRendererRegistry(registry)` | Creates a typed React renderer registry.                                      |

| `useEzFormInstance()`                   | Reads the nearest internal form instance context.                             |

| ----------------------- | --------------------------------------------------------------- |
| `ReactRendererProps`    | Props passed to a React field renderer.                         |
| `ReactRenderer`         | React component type for a field renderer.                      |

## End-To-End Core Example

```ts twoslash
import {
  defineForm,
  deserializeForm,
  FieldDataType,
  serializeForm,
  toTanStackOptions,
  toValibotSchema,
  ValidationRuleKind,
} from '@easyform/core';

const form = defineForm({
  fields: {
    email: {
      type: FieldDataType.String,
      defaultValue: '',
      rules: [{ kind: ValidationRuleKind.Email }],
    },
  },
});

const stored = JSON.stringify(serializeForm(form));
const restored = deserializeForm(stored);
const schema = toValibotSchema(restored);
const options = toTanStackOptions(restored);
```
