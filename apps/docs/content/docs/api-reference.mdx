## Core

[Open public exports on GitHub](https://github.com/VueEasyForm/tgb-form/blob/main/packages/core/src/index.ts).

### Functions

| API                                 | Description                                                                     |
| ----------------------------------- | ------------------------------------------------------------------------------- |
| `defineForm(definition, options?)`  | Parses, validates, normalizes, and clones a form definition.                    |
| `serializeForm(form)`               | Returns JSON-safe form data and removes runtime-only registries.                |
| `deserializeForm(input, options?)`  | Parses a JSON string or unknown value into a normalized runtime form.           |
| `VFormDefinition`                   | Valibot schema for a full form definition.                                      |
| `VFieldDefinition`                  | Valibot schema for a single field definition.                                   |
| `cloneJson(value)`                  | Deep-clones JSON-safe data.                                                     |
| `isJsonObject(value)`               | Checks for a plain JSON object, excluding arrays and `null`.                    |
| `toValibotSchema(form)`             | Compiles a form into a Valibot object schema.                                   |
| `createValidatorRegistry()`         | Creates a registry for named custom validator compilers.                        |
| `createRendererRegistry(registry)`  | Creates a named and type-based renderer registry while preserving literal keys. |
| `resolveRenderer(field, registry)`  | Resolves `field.component` first, then `registry.byType[field.type]`.           |
| `getDefaultValues(form)`            | Extracts cloned default values from top-level fields.                           |
| `toTanStackOptions(form, options?)` | Returns `defaultValues` and `validators.onSubmit`.                              |

`defineForm` and `deserializeForm` accept optional runtime registries. `validators` are consumed by `toValibotSchema` and `toTanStackOptions`. `renderers` are carried for adapter/bootstrap code and for component-name type narrowing; core validation helpers do not depend on them.

[Open TanStack helpers on GitHub](https://github.com/VueEasyForm/tgb-form/blob/main/packages/core/src/tanstack.ts).

```ts twoslash
import { defineForm, deserializeForm, serializeForm } from '@tgb-form/core';
import { createValidatorRegistry, toValibotSchema } from '@tgb-form/core';
import { createRendererRegistry, resolveRenderer } from '@tgb-form/core';
```

`createValidatorRegistry()` supports a fluent registration style:

```ts twoslash
import * as v from 'valibot';
import { createValidatorRegistry } from '@tgb-form/core';

const validators = createValidatorRegistry()
  .register('startsWithA', ({ message }) =>
    v.check((value: string) => value.startsWith('A'), message),
  )
  .register('endsWithZ', ({ message }) => v.check((value: string) => value.endsWith('z'), message));
```

| API                                     | Description                                                                   |
| --------------------------------------- | ----------------------------------------------------------------------------- |
| `TgbFormContext`                        | Provides a React renderer registry to descendant forms and fields.            |
| `TgbForm`                               | Creates or accepts a TanStack React Form instance and renders ordered fields. |

| `createReactRendererRegistry(registry)` | Creates a typed React renderer registry.                                      |

## End-To-End Core Example

```ts twoslash
import {
  defineForm,
  deserializeForm,
  FieldDataType,
  serializeForm,
  toTanStackOptions,
  toValibotSchema,
  ValidationRuleKind,
} from '@tgb-form/core';

const form = defineForm({
  fields: {
    email: {
      type: FieldDataType.String,
      defaultValue: '',
      rules: [{ kind: ValidationRuleKind.Email }],
    },
  },
});

const stored = JSON.stringify(serializeForm(form));
const restored = deserializeForm(stored);
const schema = toValibotSchema(restored);
const options = toTanStackOptions(restored);
```
